Documentazione dei tipi enumerati

enum HTTPRequestData_flags

I valori di questa enumerazione vengono usati per creare una bitmask che rappresenta informazioni e istruzioni di operazione per la richiesta HTTP a cui si riferiscono.

Valori del tipo enumerato
HTTPRequestData_flags_stop_parsing 

Se questa opzione è settata, l'elaborazione della richiesta deve terminare.

HTTPRequestData_flags_keep_alive 

Se questa opzione è settata, la connessione non deve terminare dopo l'invio della risposta.

HTTPRequestData_flags_dont_send_payload 

Se questa opzione è settata, devono essere inviati solo gli header della risposta.

HTTPRequestData_flags_get_error_page 

Se questa opzione è settata, al posto del file richiesto il server deve inviare una pagina di errore.
